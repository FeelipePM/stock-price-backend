## Clone the project

```bash
$ git clone https://github.com/FeelipePM/stock-price-backend.git
```

## Go to the project directory

```bash
$ cd stock-price-backend
```

## Install dependencies

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Technologies

- [NestJS](https://nestjs.com/)
- [Typescript](https://www.typescriptlang.org/)
